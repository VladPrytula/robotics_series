# Web Research Notes (For Proposal Drafting)

This file records the sources used to ground the Manning proposal draft.

## Manning proposal form

Source: `https://www.manning.com/write-a-book/proposal` (downloads `ManningBookProposal.docx`)

Key sections in the template (abridged):
- "Tell us about the book's topic" including "Why is it important now?"
- "Tell us about your readers" (minimum skills, job roles, motivations)
- "Tell us about the competition and the ecosystem" (books, videos, websites, communities)
- "Table of contents" guidance for an In Action style book (chapter objectives, summaries, numbering)
